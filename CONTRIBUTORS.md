## ğŸ¯ How to Get Added

Want to see your name here? Here's how:

1. **Make a contribution**
   - Fix a bug
   - Add a feature
   - Improve documentation
   - Report a detailed bug with reproduction steps
   - Improve test coverage

2. **Get your PR merged**
   - Follow the guidelines in [CONTRIBUTING.md](docs/CONTRIBUTING.md)
   - Pass code review
   - Have your pull request merged
